# CV-Booster-PRO 🚀

**AI-enhanced CV rewriting app**  
Improve clarity, tone, and structure of your CV instantly.

## 💡 Features

- Paste your raw CV
- 1-click rewrite using AI
- Stripe payment link integration (£15 per rewrite)
- Clean, responsive UI built with React + Vite

## 🛠 Tech Stack

- React 18
- Vite
- Stripe
- Hosted on Vercel
- GitHub for source control

## 🧠 How It Works

1. Paste your CV in the text area
2. Click “Boost My CV – £15”
3. App rewrites your CV for clarity and professionalism
4. Pay securely via Stripe
